Особенностью классов оберток является то, что у них есть автоупаковка и автораспоковка. 
1. byte -> Byte
2. short -> Short
3. int -> Integer
4. long -> Long
5. float -> Float
6. double -> Double
7. char -> Character
8. boolean -> Boolean

```java
int num = 5; Integer numObj = num; // Автоупаковка 
Integer anotherNumObj = 10; 
int anotherNum = anotherNumObj; // Распаковка
